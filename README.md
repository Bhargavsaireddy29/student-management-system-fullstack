Student Management System

A full-stack Student Management System built using React on the frontend and Spring Boot on the backend. The application supports secure student data handling with complete CRUD operations, pagination, and authentication.

Features

Full-stack development using React (Axios, Routing) and Spring Boot (REST APIs, JPA, MySQL)

Complete CRUD operations, pagination, and global exception handling

JWT-based authentication and role-based authorization

Centralized Log4j2 logging

Email notifications

Dockerized deployment for frontend, backend, and MySQL

Microservice-style architecture with multi-stage Docker builds

Nginx-served React UI

Clean 3-layer backend architecture (Controller – Service – Repository)

Tech Stack

Frontend: React

Backend: Spring Boot, JPA

Database: MySQL

Deployment: Docker, Nginx
